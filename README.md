## Getting Started

1. Clone the repository
```bash
git clone [your-repo-url]
cd crypto-dice-game
```

2. Install dependencies
```bash
npm install
```

3. Start the development server
```bash
npm run dev
```

The application will be available at `http://0.0.0.0:5300`.
